Get the starting date and time of the sporting events.

== Description ==

Get the starting date and time of the sporting events. Currently you can get events from NBA, English, Spanish, German, Italian and French football leagues and cups.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload whattimeisthe folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the plugin to the sidebar.
